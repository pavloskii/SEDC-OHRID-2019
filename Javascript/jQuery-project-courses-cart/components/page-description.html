SEDC Courses

This page is a demo project for the purposes of the SEDC Academy Course: Javascript Advanced. For the making of this website it was used:

  1. jQuery
  2. Bootstrap
  3. Font-Awsome Icons

The page consists of a home page, where the user can find the most popular courses and some info about the whole page. There is also a Courses page where the users can find all the courses and use the search and sorting filters. Also every bought item goes to the cart page.
